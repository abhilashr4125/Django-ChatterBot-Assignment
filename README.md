# Django ChatterBot Assignment

## Overview

This project demonstrates a simple Question & Answer chatbot built using Python, Django, and ChatterBot.

## Technologies Used

- Python 3.10
- Django
- ChatterBot
- SQLite

## Installation

Create a virtual environment

```bash
python -m venv venv
```

Activate it

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Train chatbot

```bash
python chatbot/train.py
```

Run chatbot

```bash
python chatbot/chat.py
```

Example

```
You: Hello
Bot: Hi

You: How are you?
Bot: I am doing well.
```
